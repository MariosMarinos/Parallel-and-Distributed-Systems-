Trexontas to programma o pelatis exei 3 aitimata stin diathesi tou.
To proto einai gia na diabasei mia sygkekrimeni ptisi me sygkekrimeno kodiko.
kai to aitima tha exei tin morfi READ (code) kai o server stelnei piso 
oles tis plirofories tis sygkekrimenis ptisis.To 2o aitima einai gia tin
eisagogi mias kainourias ptisis kai exei tin morfi WRITE (code) (ANAXWRISI/AFIKSI) (time)
kai stelnei piso WOK an i egraffi itan epityxis, akomi an to code yparxei idi sto map tote
to kanei overwrite.Telos, me to aitima REMOVE (code) elegxei an yparxei ptisi me ayton ton
kodiko kai an yparxei tote tin afairei kai epistrefei ston pelati SUCCESSFULLY REMOVED.
